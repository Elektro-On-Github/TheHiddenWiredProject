
### Edit the Tor configuration:

```bash
nano /etc/tor/torrc
```

Add:

```

HiddenServicePort 80 127.0.0.1:80
```

Then restart Tor:

```bash
systemctl restart tor
```

Get your .onion domain:

```bash
cat /var/lib/tor/serverTHWP/hostname
```
